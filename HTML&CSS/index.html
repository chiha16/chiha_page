プログラミングに必要な機材

まずプログラミングとは、ある特定のコンピューティングの結果を得ることを目的として、実行可能なコンピュータープログラムを設計・構築するプロセスのことである。（ウィキペディア）
1.PC　一番最初はわかると思いますがパソコンが必要です。性能でいうとウェブプログラミングぐらいなら家電量販店で売っているようなノートPCでもできます。またunityなどで作るゲームにはゲーミングPCなどの高性能なPCが必要です。
2.必須なソフトウェア　これがないとプログラムを書いたり書いたプログラムを実行できません。まずクロームをおすすめします。もともと入っているものでもいいんですが、僕はクロームを使っています。次にテキストエディタはBracketsをおすすめします。メモ帳などでも作成できますがこっちのほうが何十倍も便利です。

ソフト立ち上げからファイルの作成の仕方

まずインストールしたBracketsを開き右上にある「開く」をクリックします。そこから自分が作ったフォルダを選択し開いてくださいフォルダ名はアルファベットなら何でもいいです。
次にファイルを作成します。右にある灰色の枠で右クリックしファイルを作成から作成してください。ファイル名は必ず「index.html」にしてください。

作成したらこちらを打ってください。

2行改行したら真ん中にマウスポインタを合わせて下の文章を打ってください。ヘッドとはこのサイトの周りを整えるようなものです。例えばCSSを読み込んだりなど、タイトルはサイト上に出るものです。

＜head＞
＜title＞“好きなタイトル”＜/title＞
＜/head＞

次にボディータグをつけます。これは文字や画像などをつけるのに必要なタグです。

＜body＞

色々なタグをつけよう

テキストタグの種類 タグの使い方・意味
ｐタグ 段落など主に文字を書く
ｈ1～6 見出しなど大きな文字を書きたいときに使う、番号が小さいほど文字は大きくなる
div 文字以外にも使うことがある

テキストタグについてはこのくらいでしょうかもっといっぱいあると思いますが僕が知ってしるのはこのくらいです。まだまだ調べると出てくるので調べてみてください。またHTMLタグ一覧などで調べると他のタグやそれを使った応用とかも出てきます。
